Service Client

is a example service client.

Prerequisites:
- .NET SDK 5.0


=====================================
Development Environment
=====================================

Service Client application:
- On any terminal move to the "ServiceClient" folder (the folder containing the "ServiceClient.csproj" file) and execute these commands:

dotnet restore

MongoDB Client application:
- On any terminal move to the "MongoDBClient" folder (the folder containing the "MongoDBClient.csproj" file) and execute these commands:

dotnet restore
